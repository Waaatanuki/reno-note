# 关于

收集整理装修资料。

## 提示词

我们即将对一套老房进行全包装修，已与装修公司确认初步清单，因签订合同后变更需额外付费，现需一份覆盖老房装修特殊需求 + 全包装修全流程的完整检查点清单，要求尽可能详细全面。请清单包含但不限于：1. 老房基础改造（墙体拆改、水电管线更新、墙面 / 地面基层处理、门窗拆换、防水工程等）；2. 全包装修核心项目（主材 / 辅材品牌型号规格确认、施工工艺标准、安装验收要求）；3. 功能区域细节（客厅、卧室、厨房、卫生间、阳台等各空间专属需求，如厨卫排水、储物布局、插座预留等）；4. 隐蔽工程重点（水电布线、吊顶龙骨、墙体挂网等隐蔽项的材料 + 施工 + 验收）；5. 合同关联项（工期节点、付款比例、质保范围、售后承诺、增项界定标准）；6. 老房专属隐患排查（墙体开裂处理、管道老化更换、隔音 / 保温补充、承重结构保护等）。请按逻辑分类梳理检查点，每个项目明确‘需确认内容 / 避免遗漏点’，确保无关键环节缺失，方便我们逐一核对装修公司清单。
