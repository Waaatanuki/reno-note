# 需求文档

整体

- 地暖、壁挂空调、不要吊顶、只需要石膏线

玄关

- 鞋柜、衣帽柜、能放一些小杂物的空间、全身镜

厨房

- 开放式
- 预留垃圾桶的位置
- 电器包括：双开门冰箱、水槽一体洗碗机、微波炉、咖啡机、蒸烤一体机、空气炸锅、燃气热水器

卫生间

- 尽量三分离
- 智能马桶、自动上下水扫地机器人、堆叠的洗衣机和烘干机

客厅

- 4人餐桌、餐边柜+收纳展示柜、3人宽度沙发、灵活一点不用太大的茶几、壁挂电视、电视柜、一个台式电脑桌、健身区

主卧

- 单独的衣帽间，折叠门
- 1.8米床、两侧通行、床头柜
- 梳妆台

阳台

- 鞋柜、晾衣晒被子

次卧

- 1.5米床、一个书桌、一个衣柜，不需要单独卫生间

杂物间

- 无障碍进出（行李箱、婴儿车等可以直接推入）

装修手帐

看出图后的需求

- 入户玄关置物修改
- 厨房燃气热水器能放马桶间
- 厨房咖啡柜最左靠近客厅侧做一列镂空，朝向客厅
- 厨房咖啡柜台面上加一层隔板上，放微波炉
- 餐厅餐边柜吊柜下一个开放格放杯子
- 餐厅餐桌区灯位不装灯但是想先开一个洞
- 客厅电视柜悬浮安柜腿防掉落，下面有支撑距地面待定厘米说完
- 客厅电视柜墙后预留管道不留明线
- 客厅沙发后面放几个层架，有层次感
- 主卧阳台烘干晾衣架
- 主卧衣帽间做成通顶门
- 储藏间不用外开柜了，做成三折叠通顶门
- 干区洗脸盆下浴室柜做成无底柜，可能的话要个抽屉
- 干区洗脸镜柜做成无底柜
- 湿区淋浴会不会水从门缝漏出来

设计需求

- 客厅样式
- 厨房不安装蒸烤一体机
- 厨房垃圾桶放在哪
- 主卧客厅都使用风管机，解决直吹和外机占位问题
- 一些细节的储物点
- 餐边柜450，电视柜
- 电视柜悬浮加支架，离地面的高度能让扫地机器人进去
- 解决入户玄关、厨房、卫生间昏暗问题
- 包竖着的暖气管时用能磁吸的材质
- 主下水管检修口留在
- 轨道灯线形灯明装
- 浴室墙面pvc墙板，地面用地胶
- 入户门打开方向满意
- 餐厅餐边柜地柜一列下面放水桶上边做抽屉，一列做可活动层板放箱装饮料

装修疑问

- 墙面垂平还是顺平
- 窗台石包含么
- 踢脚线、过门石是否单算
- 我们加的灯怎么算
- 浴室刨地加不加钱

收纳需求

- 玄关及延长线：l300*w300*h250室内鞋3双，另4双备用鞋，1100外套4件，钥匙眼镜口罩纸巾等杂物，帆布包，冬季帽子围巾手套2套，雨伞雨衣。入户灯，感应灯，镜前灯，采光。头盔临时放置处。
- 厨房按大小排序：小家电（净水机，直饮机，咖啡机及咖啡用品，电蒸锅，微波炉，空气炸锅，电饭煲，电火锅，破壁机，真空机，面包片机、早餐机）。锅类（平底锅、煮面锅、大炒锅、珐琅锅、微压锅、大蒸锅）。垃圾桶，扫把簸萁。常温需加工食品，常温蔬菜，米桶杂粮桶，桶装油。备菜盆，大盘子大碗，饭盒调料架，案板2个，锅盖2个，刀架，擦丝器蒜蓉器，厨房纸品，保鲜膜袋，塑料袋，其他备品。主灯，咖啡区灯，切菜洗菜炒菜灯，排风扇。珐琅板收纳。
- 卫生间：扫地机器人。马桶区（参考日本那种，卷筒卫生纸囤货）。换衣区（毛巾、挂衣、脏衣篮）。淋浴区（淋浴用品、刮水用品、泡脚、排风、浴霸）。洗脸区（牙杯倒挂，洗脸巾，用后的洗脸巾，吹风机，插电充电，洗面奶水乳，指甲钳棉签等小件，备品）。
- 洗衣区（下洗，中间抽出熨衣，洗衣液等，梯子）。主灯，镜前灯。
- 办公区：1000电脑桌*2，250+主机*2，电脑区收纳背板，电竞椅。沙发侧额外需要放置书处；电视柜侧垃圾桶】。户外摄像头。
- 客厅多功能公区：65寸电视，wifi机顶盒，游戏机，健身器材（800瑜伽垫、350健身环），桌游（330拼图、260砖块），周边，公用药口罩，充电器区。吊扇灯。
- 餐厅餐边柜区：煮茶煮水设备，常温饮料2箱，零食，杯子水壶，常温水果，纸杯纸盒纸袋，用餐杂物。餐边柜灯，餐桌灯。
- 次卧：投影仪，沙发床，预留电脑桌区
- 杂物间：行李箱3个，露营桌椅，露营车，烧烤架，车载冰箱，纸品囤货，戴森及配件，电风扇，椅子，汽车摩托电动车用具，食品囤货，洗漱囤货，礼品，纸和打印机。密室门，装饰灯。
- 主卧：衣帽间（应季衣服，收纳衣服，收纳冬天被子，包包，鞋子，重要证件）。化妆台（书，化妆品，化妆品备货，饰品）。电器（除螨仪，加湿器，空气净化器）。床头柜2个。霄私人柜。禛私人柜。吊扇灯。
- 阳台：晾衣架，窗帘。
